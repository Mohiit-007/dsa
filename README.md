# 🧠 DSA-CPP

> A structured collection of **Data Structures & Algorithms** implemented in **C++** — built for placement prep and deep conceptual mastery.

## 📁 Repository Structure

| Folder | Topic | Description |
|---|---|---|
| `ARRAY` | Arrays | 1D array problems — traversal, prefix sums, sliding window |
| `2DARRAY` | 2D Arrays | Matrix operations, spiral traversal, rotation |
| `CHAR ARRAY` | Character Arrays | String manipulation using char arrays |
| `BASIC MATH` | Mathematics | GCD, LCM, prime checks, modular arithmetic |
| `BASICOFCPP` | C++ Basics | Fundamental C++ syntax and concepts |
| `POINTER` | Pointers | Pointer arithmetic, double pointers, references |
| `OOPS` | Object-Oriented Programming | Classes, inheritance, polymorphism, encapsulation |
| `STL` | Standard Template Library | vectors, maps, sets, pairs, priority_queue |
| `PATTERN` | Patterns | Star/number patterns — loop logic practice |
| `RECURSION` | Recursion | Base cases, backtracking, recursive tree problems |
| `DnC` | Divide & Conquer | Merge sort, quick sort, binary search variants |
| `SEARCH` | Searching | Linear search, binary search, search on answer |
| `SORT` | Sorting | Bubble, selection, insertion, merge, quick sort |
| `LINKEDLIST` | Linked List | Singly/doubly linked lists, reversal, cycle detection |
| `STACK` | Stack | Stack using arrays/STL, monotonic stack problems |
| `QUEUE` | Queue | Queue using arrays/STL, deque, circular queue |
| `BINARYTREE` | Binary Tree | Traversals, diameter, height, LCA |
| `BINARYSEARCHTREE` | BST | Insert, delete, search, BST validation |
| `HEAP` | Heap | Min/Max heap, heap sort, priority queue problems |
| `GRAPH` | Graph | BFS, DFS, Dijkstra, cycle detection, topological sort |
| `TRIES` | Tries | Trie insert/search, prefix matching |

## 🚀 Getting Started

### Prerequisites
- A C++ compiler (g++ / clang++)
- C++17 or later recommended

### Clone the Repository
```bash
git clone https://github.com/Mohiit-007/DSA-CPP.git
cd DSA-CPP
```

### Compile & Run any file
```bash
g++ -o output ARRAY/filename.cpp
./output
```
